DecisionTreeGP run instructions

From the repository root:

1. Compile:
   javac "Assignment 3/DecisionTreeGP.java"

2. Train:
   java -cp "Assignment 3" DecisionTreeGP train

3. Test:
   java -cp "Assignment 3" DecisionTreeGP test

Notes:
- Training reads Assignment 3/Breast_train.csv and Assignment 3/Breast_test.csv.
- Training saves the model to Assignment 3/decision_tree_gp.model.
- Testing loads the saved model and evaluates it on the test data.
